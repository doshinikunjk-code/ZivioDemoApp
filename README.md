# Zivio × Desi Road — AI Ordering Demo

Live demo of Zivio AI Operations Platform built for Desi Road Restaurant, Brampton.

## What it does
- AI WhatsApp ordering in English, Punjabi, and Hindi
- Natural Indian voice powered by ElevenLabs
- Live Claude Haiku AI — type any order, ask anything
- Demo Call — hear AI answer as Desi Road in Punjabi
- No prices shown unless customer asks
- Family ordering — multiple people, one order
- Kitchen alert on confirmation
- Works offline with cached responses

## Built with
- Claude Haiku 4.5 (AI brain)
- ElevenLabs (Indian voice)
- Express.js (server)
- Deployed on Railway
